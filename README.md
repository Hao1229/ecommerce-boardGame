# 電商桌遊網站練習

![](https://i.imgur.com/iMg2xU4.png)

## Demo
https://hao1229.github.io/ecommerce-boardGame/#/

## 前後台介紹
### 前台
* Navbar 購物車與優惠卷即時更新，方便使用者隨時查看
* 產品列表可根據產品分類顯示不同種類商品
* 單一產品頁面，具有更詳細的商品介紹
* 購物車詳細頁面，可套用優惠卷做價格的即時更新
* 結帳確認頁面具有結帳成功的回饋

### 後台
* 主頁具數據統計
* 產品頁可新增或修改產品內容
* 訂單頁可以確認顧客訂單與詳細資料
* 優惠卷頁可以新增或修改優惠卷

## 使用技術
* Vue Cli
* Vue Router
* SCSS
* RWD
* Bootstrap 4
* ESlint(Standard)

## 使用Plugin
* VeeValidate
* Vue axios
* vue-loading-overlay 
* Vue-Chart.js
